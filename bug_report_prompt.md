**Аббревиатуры платформ и не только:**

**[Web App]** - CRM / админка / бэк офис для работы на персональных компьютерах https://app.cleverone.tech/

**[iOS Biz] -** бизнес приложение/админка/бэк офис для работы на iPhone

**[Android Biz] -** бизнес приложение/админка/бэк офис для работы на телефонах Android

**[Web Site] -** сайт клиентов (MERCHANT) для работы с ним на персональных компьютерах 

**[Web Site | Booking]-** сайт по бронированию для работы с ним на персональных компьютерах 

**[Web Site Mobile Web] -** сайт для работы с ним на мобильных устройствах

**[iOS Merchant] -** мобильные приложения для Клиентов на iPhone

**[Android Merchant] -** мобильные приложения для Клиентов на Android

**[Integration] -** задачи связанные с интеграциями

**[Design] -** задачи связанные с дизайном

**Merchant -** тип бизнеса, который был выбран при создании: Ресторан, Поставщик, Ресторан Хаб, Магазин и прочие

**Environment -** это окружение, cреда на которой проводилось тестирование, версия OS (операционной системы, в настройках телефона), название модели девайса

**App Version** - это версия приложения и среда разработки Dev / Life

**Precondition** **(при необходимости)** - предусловие при котором воспроизводиться описываемый баг / задача
