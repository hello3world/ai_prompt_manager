Нужно создать программу которая предоставляет веб интерфейс для отправки краткого описания баг и получения по API AI подробного баг репорта на основании инструкции переданной
Подробное описание:
1. Веб страница с textarea для ввода описания бага, все то о чем известного и кнопка получить баг репорт
2. Далее логика берет текст из текстового поля и подставляет в PROMPT, который отправяляется к AI
3. Промпт состоит из запроса и передаваемым с ним инструкцией  
4. Инструкция находится в bug_report_prompt.md
5. Ответ выводится целиком на web странице
6. Креды для подключения к qwen api находятся в .env.prod
7. используй модель qwen max
8. Запускать хочу через докер компоуз


docker compose up --build
